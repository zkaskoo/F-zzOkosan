---

## Problem Statement

Users struggle with:
1. Managing recipes from multiple sources
2. Creating shopping lists manually
3. Combining ingredients from multiple recipes
4. Converting between different units (dkg, ek, csésze, etc.)

---

## Solution

A web application that:
- Allows users to share recipes (Instagram-style)
- Automatically parses ingredients using AI/NLP
- Generates optimized shopping lists
- Combines ingredients from multiple recipes
- Supports weekly meal planning
